# 시스템학원 홈페이지

## 실행
1. 프로젝트 루트에서 서버 실행
   - `python3 -m http.server 4173`
2. 브라우저 접속
   - 메인 홈페이지: `http://localhost:4173/index.html`
   - 스네이크: `http://localhost:4173/snake.html`
   - 갤러그: `http://localhost:4173/galaga.html`
   - 벽돌깨기: `http://localhost:4173/breakout.html`

## 메인 페이지 구성
- 학원 소개 Hero
- 과목 안내: 영어 / 수학 / 과학
- 상담 및 운영 시간 안내
